# MVC Student Project

## Description

ASP.NET Core MVC application for managing student data.
The project demonstrates CRUD operations, working with a database through a repository pattern, and communication with an external API that provides JSON data and predictions.

The application is ready to deploy on Azure, which has already been successfully done.

## Tech Stack

* ASP.NET Core MVC
* C#
* SQL Server
* REST API
* Azure App Service
